# Climbing Calendar

A mobile-friendly web application and REST API for organising climbing sessions, finding climbing partners, and connecting with other climbers.

## Overview

Climbing Calendar helps climbers coordinate sessions without relying on scattered group chats and social media posts.

Users can create climbing plans, share their availability, find belayers, and join sessions organised by other climbers.

The goal is to make it EASY for climbers to answer:

- Who is climbing tonight?
- Does anyone need a belayer on Saturday?
- Is anyone free for a session after work?
- Are there any group sessions happening this week?

## Features

### Authentication

- User registration
- User login and logout
- Token-based authentication using Laravel Sanctum

### Climbing Sessions

- Create climbing sessions
- View upcoming sessions
- Update or cancel plans
- Specify location, date, and time

### Groups/Communites

- Join local climbing communities
- Find climbing partners
- Request a belayer
- Join existing sessions
- View who is attending

### Mobile Friendly

- Designed to work well on both desktop and mobile browsers
- REST API for future mobile application development
